# first-repo Learning Docs

このフォルダは、GitHub学習の進捗を引き継ぐためのドキュメント置き場です。

## ドキュメント一覧
- `docs/learning-plan.md`: 学習計画と到達目標
- `docs/session-log.md`: 実施ログ（何をやったか）
- `docs/command-cheatsheet.md`: よく使うコマンドの最小チートシート

## 使い方
1. 学習開始前に `learning-plan.md` の「次にやること」を確認
2. 学習後に `session-log.md` に1セッション分を追記
3. つまずいたコマンドは `command-cheatsheet.md` に追記
